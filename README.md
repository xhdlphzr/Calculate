# Calculate

## Version 0.2.0

Calculator using command line

## Instruction

### Input Sample

```
1 + 2
3 - 1
3 * 2
3 / 8
```

### Output Sample

```
3
2
6
0.375
```

### Explain

You can input a + b or a - b or a \* b or a / b to calculate a + b or a - b or a \* b or a / b
